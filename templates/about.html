About Us

A Brief Introduction

Hello and welcome to our site!


This is an interactive web-application revolving around visualisations and data exploration, created for a school project. The project was done using the scrum-method, meaning we did 4 sprints and sprint reviews etcetera. We learned a lot from this as it was the first big project for some of us and it gave us some experience which we appreciate.
